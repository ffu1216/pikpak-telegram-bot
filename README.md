## マグネットリンクをテレグラムボット経由で保存する方法

これを知りたかった人も多いはずなので、丁寧に書いておく。

PikPak 公式の仕様では、テレグラムボットへのマグネットリンクは **Deeplink 形式** を使う。具体的には以下のような URL になる：


https://t.me/pikpak_bot?start=magnet_[INFO-HASH-STRING]


例えば、マグネットリンクが：


magnet:?xt=urn:btih:c12fe1c06bba254a9dc9f519b335aa7c1367a88a


だった場合、`urn:btih:` 以降の部分（info-hash）を取り出して：


https://t.me/pikpak_bot?start=magnet_c12fe1c06bba254a9dc9f519b335aa7c1367a88a


という形の Deeplink を作成する。このリンクをタップすると `@pikpak_bot` が起動し、コンテンツを PikPak に保存する流れになる。

**なぜこの形式？**

テレグラムボットのディープリンクには「64 文字以内、特殊文字（:、? など）不可」という制限があるため、完全なマグネット URI をそのまま使うことはできない。info-hash だけを抽出して Deeplink に変換する必要がある。

もちろん、PikPak アプリやウェブ版でマグネットリンクを直接入力する方法もある。テレグラム経由にこだわらないなら、アプリから直接貼り付けるほうが手軽だ。

**Q：複数の PikPak アカウントをテレグラムと連携できますか？**
